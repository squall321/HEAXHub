Static HTML 데모 static_html

이 페이지는 HEAXHub 의 static_html 스택 시그니처를 시연하는 단일 파일 정적 페이지입니다. 별도 런타임 없이 Caddy 의 file_server 가 그대로 서빙합니다.

Stack 정보 MANIFEST

.portal/manifest.yaml 의 빌드 시그니처입니다.

build.stack · static_html
build.root · .
launch.mode · static
app_type · web_app

이 페이지가 뭐가 빌드되었나 NO-BUILD

static_html 스택은 아무 것도 빌드하지 않습니다. 아래는 명시적으로 일어나지 않는 것들:

  • venv 생성 없음 — Python 인터프리터/패키지 설치를 하지 않음
  • npm/pnpm install 없음 — Node 의존성 다운로드 단계 없음
  • build 단계 없음 — webpack/vite/next build 가 돌지 않음
  • 런타임 프로세스 없음 — uvicorn/node/streamlit 데몬이 뜨지 않음
  • health check 폴링 없음 — 살아있는 프로세스가 없으므로

HEAXHub 는 그저 build.root 디렉터리를 Caddy 의 file_server 에 마운트만 해 줍니다.
