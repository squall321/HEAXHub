This is a pre-built MkDocs documentation site bundled as a HEAXHub integration. The output under site/ is exactly what mkdocs build produces locally.

Sections

  • API Reference — endpoint and schema docs.

Why static?

The portal serves these HTML files via Caddy's file_server directive. No Python runtime is needed at serve time — the MkDocs toolchain only runs at build time on the operator's machine.
