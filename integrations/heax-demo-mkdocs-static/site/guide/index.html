User Guide

Welcome to the demo guide. This walkthrough covers the three common operations operators perform on this integration.

1. Install

Register the integration through the HEAXHub portal. The pre-built site/ directory is copied to the runner.

2. Configure

Edit mkdocs.yml in the source repository, then run mkdocs build to regenerate this directory.

$ mkdocs build --clean --strict

3. Serve

Caddy is started by HEAXHub with a generated config that points at site/ and exposes it under the integration's route.
